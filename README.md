# pa11 LV-NCU
New LV-NCU Livery for Awall86's Piper Pa11 for Flightgear.
https://www.github.com/awall086/pa11/

Uncompress and copy the Aircraft folder into your Flightgear pa11 folder.
For example, on Linux could be done with something like:

<code>cp -r Aircraft/pa11/ $USER/.fgfs/Aircraft/org.flightgear.fgaddon.stable_2024/Aircraft/pa11</code>
